# charts — профессиональные чертежи/графики (Python + Matplotlib)

Воспроизводимые технические рисунки кодом. Matplotlib экспортирует **PDF (вектор) + SVG + PNG** — годится и для заявки, и для презентаций, и для доводки в Inkscape/Illustrator.

## Как запускать (Windows)
Python установлен в пользовательский профиль (winget, без админ-прав):
```
%LOCALAPPDATA%\Programs\Python\Python312\python.exe <скрипт>.py
```
Зависимости: `matplotlib`, `numpy` (уже стоят). Установка новых: `python.exe -m pip install <пакет>`.

## Содержимое
| Скрипт | Что рисует | Данные / модель |
|---|---|---|
| `hydrograph_peak_shaving.py` | гидрограф срезки пика паводка (R-02) | линейный резервуар S = K·Q |
| `site_plan_maly_taldykol.py` | **геопривязанный план площадки** (озеро + вмешательства на спутнике) | контур из OpenStreetMap (osmnx) + подложка Esri (contextily); вмешательства от геометрии озера |

> `site_plan_maly_taldykol.png` также скопирован в `../../07_Презентации/` и встроен в слайд 2 презентации «Концепт_Малый-Талдыколь».
> Геозависимости (стоят): `osmnx`, `contextily`, `geopandas`, `shapely`, `pyproj`. Нужен интернет (OSM Overpass + тайлы Esri).

## Как масштабировать
- Новый чертёж → новый `.py` по образцу (единый стиль: палитра teal/red, шрифт DejaVu Sans — кириллица ок).
- Следующие кандидаты: масштабные поперечные разрезы (R-01 болото, R-05 иловое поле), баланс «приток/сброс», график качества воды.
- Геопривязанный план площадки → отдельный стек (`geopandas` + `contextily` для подложки-тайлов, либо QGIS).
