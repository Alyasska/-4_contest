## B. По конкретным решениям

### R-01 Полировка стоков КОС · R-05 Иловые тростниковые поля (constructed/treatment wetlands, reed beds)
- 📄 EPA Constructed Wetlands Design Manual (см. блок A) — основной инженерный источник.
- 🔗 **SSWM** — практические листы по «sludge drying reed beds»: <https://sswm.info/pavitr-toolkit/get-know-pavitr-technologies/water-reclamation-solutions/sludge-drying-reed-beds-for-faecal-sludge-treatment>
- 📄 **MDPI Water (open access)** — «Sludge Dewatering and Mineralization in STRB»: <https://www.mdpi.com/2073-4441/9/3/160>
- 🔒 Обзор очистных болот в **холодном климате** (важно для Астаны) — Springer 2024: <https://link.springer.com/article/10.1007/s11356-024-34065-4> (абстракт открыт)
- 🔗 **ARM Reedbeds** (подрядчик, кейсы по иловым полям): <https://armreedbeds.co.uk/projects/sludge-treatment/>

### R-02 Паводковый буфер · R-07 Грунтовые воды (floodplain/stormwater, aquifer recharge)
- 📄 **NNBF Guidelines** (см. блок A) — главный методический документ.
- 🔗 **Qunli Stormwater Park** (Харбин, климат как Астана) — кейс ASLA 2012: <https://www.asla.org/2012awards/026.html> · проектная страница Turenscape: <https://www.turenscape.com/en/project/detail/435.html>
- 🔒 **Johnson Creek / Портленд** (срезка пика −25%) — статья Ahilan 2018: <https://onlinelibrary.wiley.com/doi/full/10.1111/jfr3.12251>
- 🔗 **Naturally Resilient Communities** — практ. кейсы по восстановлению пойм: <https://nrcsolutions.org/restoring-floodplains/>

### R-03 Подавление пыли на осушённом ложе (модель Owens Lake) — самый документированный аналог
- 📄 **Научный отчёт по эффективности мер** (GBUAPCD/Owens Lake Scientific Advisory Panel, 2020): <https://gbuapcd.org/OwensLake/OLSAP/2020_OLSAP_FINALREPORT.pdf>
- 📄 **Final EIR Phase 9/10** (LADWP) — инженерная и экологическая документация: <https://www.ladwp.com/sites/default/files/2024-03/Phase%209-10%20Final%20EIR.pdf>
- 🔗 **CA State Lands Commission (CEQA)** — пакет проектных документов: <https://www.slc.ca.gov/ceqa/owens-lake-dust-mitigation-program-phase-7a-9-10-projects/>
- 🔗 **LADWP Dust Mitigation** (обзор программы): <https://www.ladwp.com/who-we-are/ladwp-eastern-sierra/owens-lake/owens-lake-dust-mitigation>

### R-06 Водно-болотный парк (рекреация + биоразнообразие)
- 🔗 **Qunli ASLA + Turenscape** (см. R-02).
- 🔗 У **АСБК** уже есть концепция «Природный парк Талдыколь»: <https://www.acbk.kz/article/default/view?id=370>

### R-09 MRV / углерод
- 🔗 **Verra VM0033** (водно-болотные угодья — приливно-морские; для пресных степных болот методики незрелые → проверить применимость): <https://verra.org/methodologies/vm0033-methodology-for-tidal-wetland-and-seagrass-restoration/>
- Профиль и компетенции **C4** по MRV/IPCC — в `../05_Консорциум/` и `climate.kz/info about company/`.
