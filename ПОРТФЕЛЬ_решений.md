# Портфель NbS-решений (9 шт.) — с прецедентами

> Дайджест наработок. Полные версии: [материалы/исходники_md/01_КАТАЛОГ_решений.md](материалы/исходники_md/01_КАТАЛОГ_решений.md), матрица — [02_МАТРИЦА_приоритизации.md](материалы/исходники_md/02_МАТРИЦА_приоритизации.md), документация прецедентов — [03_ДОКУМЕНТАЦИЯ_прецедентов.md](материалы/исходники_md/03_ДОКУМЕНТАЦИЯ_прецедентов.md). Технические параметры/схемы — [материалы/техника/04_Технические_выжимки_и_схемы.pdf](материалы/техника/04_Технические_выжимки_и_схемы.pdf).

| # | Решение | Защищает (серый актив) | Эффект (прецедент) |
|---|---|---|---|
| R-01 | Полировочное болото для стоков КОС | очистные | TN −58%, патогены −99% (Китай/Индия) |
| R-02 | Паводковый буфер/задержание стока | ливнёвка + защита Есиля | срезка пика до −25% (Портленд); 500 тыс. м³ (Харбин) |
| R-03 | Подавление пыли на осушённом ложе | воздух/дороги/здоровье | −99,4% пыли + птицы (Owens Lake, США) |
| R-04 | Охлаждение / тепловой остров | энергоспрос/здоровье | 10–50% охлаждения в 30–200 м |
| R-05 | Иловые тростниковые поля | иловое хозяйство КОС | сухое в-во 25–40%, без химии (Сев. Европа) |
| R-06 | Водно-болотный парк | зелёная/соц. инфраструктура | Qunli (Харбин), Sydney Olympic Park |
| R-07 | Регуляция грунтовых вод | фундаменты/водоснабжение | подпитка горизонта (Харбин) |
| R-08 | Прибрежные буферы/русла | каналы/берега/КОС | фильтрация стока (Rahway, США) |
| R-09 | **MRV** (лид C4) | измеримость всего | 30% оценки гранта = MRV |

## Рекомендованный набор (по матрице)
- **Ядро:** R-02 паводковый буфер · R-03 пылеподавление · R-01 полировка КОС.
- **Витрина:** R-06 парк. **Быстрые победы:** R-08 буферы · R-05 иловые поля. **Сквозной:** R-09 MRV.

## Связь с идеями
- **Малый Талдыколь** использует R-01, R-02, R-03, R-06, R-07, R-08 (+R-09).
- **«Снежный хаб» НУ** опирается на ту же технику (отстойник + болото/биодренаж = R-01/R-08-логика) + снежное охлаждение.

> ⚠️ Числа — из реализованных проектов в других странах (ориентиры). Локальная проверка = Design-фаза.
